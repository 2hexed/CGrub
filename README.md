# CGrub
**Customize your GRUB appearance.**

# Installation

+ **Install it using**

```bash
git clone https://github.com/2hexed/CGrub.git
```

# Usage
+ **Firstly, make the file executable with:** 

```bash
chmod +x CGrub.bash
```

+ **Second, run it on your root user using:**

```bash
./CGrub.bash
```
